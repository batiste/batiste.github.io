Pricehubble React practical Test

Requisit: a modern browser, use stackblitz.com to write your code.

  • Use the latest version of React. Use latest features offered by the framework.
  • You should use all the modern features of JavaScript/TypeScript that makes sense to you, e.g. : Types, async/await, fat arrow functions, template strings, etc.
  • Focus on working and clear code rather than finishing all the steps.

Step 1

Create a page with "Github user explorer" as a title. Then add the appropriate structures that satisfy the following

  1. Use the github API to fetch data on Linus Torvalds repository (by doing an HTTP call). Here is his repositories: https://api.github.com/users/torvalds/repos.
  2. Use this data to display a list of links of his repositories using the description and the link to the github page that can be found in the API response.
  3. Do this request when the page loads the first time.
  4. Use semantic HTML where it make sense.

Step 2

Add an input field in your page that allows you to enter any username. The list of repositories should automatically update 1 second after the user has finished typing within the input. Display an error message if the username doesn't exist (404).
